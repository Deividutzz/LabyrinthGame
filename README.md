# Labyrinth Game
A small project made as a maze mini-game.
This is the source code of the actual game.

Description:

Have you ever wondered how would a small Pac-Man-like
game would be if it were introduced in Mario's universe?
Well, this might be it! An indie and interactive game that
is meant only for fun and taking some time to run from Mario's friends.
